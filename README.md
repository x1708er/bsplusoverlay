# bsplusoverlay
Completely local Streamoverlay for BS+ mod and beatleader api.

Vibe Coded with Claude Code.

Work in Progress
# Prerequisites
Python (Tested on Arch/cachyos python 3.14.3) needed as proxy for Beatleader API due to CORS.
# Usage
1. Run start.bat (windows) or start.sh (MacOS & linux)
2. Browser tab opens for settings.
3. Set http://localhost:7273/index.html as browser source in OBS.
